# Welcome to CW-Beacon + Paddle !

Hello there! This is an original project from TA2BGH.
(Link here : https://github.com/ta2bgh/cw-beacon)
This is a simple CW transmitter project based on **Arduino** and **Si5351** clock generator. 
At start it transmits a predefined message as morse code on a fixed frequency such as 7.010 MHz or any other amateur radio frequency you want between 8 kHz and 160 MHz.
And you can now connect a paddle key to transmit morse code !
